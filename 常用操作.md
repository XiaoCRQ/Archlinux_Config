# 简介

此文件用于记录可能会遇到的操作

# 内网穿透

- [下载](https://www.natpierce.cn/pc/downloads/index_new.html) 对应的版本

- 设置，注意需要sudo

- 开机自启动

  - 创建service文件 —— `sudo nvim /etc/systemd/system/natpierce.service`
  - 写入

  ```bash
  [Unit]
  Description=natpierce

  [Service]
  ExecStart=路径
  Restart=always

  [Install]
  WantedBy=multi-user.target
  ```

# 串流

- 设置镜像源 —— `sudo nvim /etc/pacman.conf`

- 写入

```bash
[lizardbyte]
SigLevel = Optional
Server = https://github.com/LizardByte/pacman-repo/releases/latest/download

[lizardbyte-beta]
SigLevel = Optional
Server = https://github.com/LizardByte/pacman-repo/releases/download/beta
```

- 下载sunshine —— `sudo pacman -S 下载sunshine/sunshine-git`

- 配置省略

## hyprland下虚拟显示器串流方案

- 任意目录创建一个sh脚本

- 写入

```bash
#!/bin/bash

W=${SUNSHINE_CLIENT_WIDTH:-2560}
H=${SUNSHINE_CLIENT_HEIGHT:-1440}
FPS=${SUNSHINE_CLIENT_FPS:-144}
NAME="SUNSHINE_V"

case "$1" in
start)
  hyprctl keyword monitor "eDP-1,disable"
  hyprctl keyword monitor "eDP-2,disable"
  hyprctl keyword monitor "DP-1,disable"
  hyprctl keyword monitor "DP-2,disable"
  hyprctl keyword monitor "DP-3,disable"

  hyprctl output create headless $NAME
  hyprctl keyword monitor "$NAME,${W}x${H}@${FPS},0x0,1"
  ;;
stop)
  hyprctl output remove $NAME
  hyprctl reload
  ;;
esac
```

- 在sunshine的应用程序中添加 `命令准备工作` —— 包含start调用和stop调用

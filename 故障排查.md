# 说明

此文当用于记录系统使用中可能会出现的问题

## 系统层面

### 无法调节亮度

其原因是 `没有背光接口` 或者在 `/sys/class/backlight` 中只存在 `nvidia_wmi_ec_backlight` 导致

- 解决方案 —— 强制启用原生GPU背光
  - `sudo pacman -S brightnessctl` 安装背光控制软件
  - `sudo nvim /etc/default/grub` 打开grub配置文件
  - `GRUB_CMDLINE_LINUX_DEFAULT="quiet splash acpi_backlight=native"` 修改这一行（加参数）
  - `sudo grub-mkconfig -o /boot/grub/grub.cfg`
  - `reboot`
  - `brightnessctl set 100%` 调节亮度，数值范围 `0-100%`

## 软件层面

### 部分软件无法使用fcitx输入法

其原因是软件没有完全适配wayland，默认启用的xwayland导致fcitx无法正常使用，修改配置即可

- 解决方案 —— 强制启用wayland
  - `cd /usr/share/applications/` 进入软件配置文件夹
  - `sudo nvim qq.desktop` 修改配置文件（linuxqq为例）
  - `Exec=linuxqq %U --enable-features=UseOzonePlatform --ozone-platform=wayland --enable-wayland-ime %U` 修改这一行（加参数）
